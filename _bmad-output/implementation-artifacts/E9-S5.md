# Story 9.5: Add Voice Match Explanations

**Epic:** E9 - Voice Agent (POST-MVP)  
**Status:** post-mvp  
**Priority:** P2  
**Dependencies:** 9.2, E5 (Matching Engine)  
**Blocks:** None

## Story

As a user,
I want to hear match explanations read aloud,
so that I can understand matches without reading.

## Acceptance Criteria

1. **Given** a match detail page
   **When** I click "Listen"
   **Then** the match explanation is read aloud

2. **Given** the audio playback
   **When** it plays
   **Then** I can pause/stop it

3. **Given** the explanation
   **When** it's read
   **Then** it sounds natural and conversational

## Tasks / Subtasks

- [ ] Task 1: Add listen button to match detail
  - [ ] Update `src/app/(protected)/matches/[id]/page.tsx`
  - [ ] Add speaker icon button
  - [ ] Show loading state while generating

- [ ] Task 2: Implement audio playback
  - [ ] Call synthesizeSpeech with explanation
  - [ ] Play audio in browser
  - [ ] Add play/pause controls

- [ ] Task 3: Cache audio
  - [ ] Consider caching generated audio
  - [ ] Avoid regenerating for same explanation

## Dev Notes

- Use ElevenLabs from 9.2
- Keep explanations concise for better audio
- Consider pre-generating audio when match is created
