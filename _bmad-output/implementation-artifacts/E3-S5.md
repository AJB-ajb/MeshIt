# Story 3.5: Build Text-Based Onboarding Page

**Epic:** E3 - Profile Management  
**Status:** ready-for-dev  
**Priority:** P0  
**Dependencies:** 3.3, 3.4  
**Blocks:** None

## Story

As a new user,
I want to describe myself in natural language,
so that I can quickly create my profile without filling forms.

## Acceptance Criteria

1. **Given** I am a new user with no profile
   **When** I am redirected after OAuth
   **Then** I see the text onboarding page

2. **Given** the onboarding page
   **When** I enter my description and click "Extract"
   **Then** I see the AI-extracted structured data
   **And** I can review before saving

3. **Given** the extracted profile
   **When** I click "Save Profile"
   **Then** my profile is saved
   **And** I am redirected to the dashboard

4. **Given** extraction errors
   **When** the AI fails to extract
   **Then** I see a helpful error message
   **And** I can try again
