# Story 5.5: Build Match Detail Page

**Epic:** E5 - Matching Engine  
**Status:** ready-for-dev  
**Priority:** P0  
**Dependencies:** 5.3  
**Blocks:** E7-S2 (Initiate contact)

## Story

As a user,
I want to see full details of a match,
so that I can decide whether to connect.

## Acceptance Criteria

1. **Given** I navigate to /matches/[id]
   **When** I am involved in the match
   **Then** I see full match details

2. **Given** the match detail page
   **When** I view it
   **Then** I see both profile and project details
   **And** I see the match score and explanation

3. **Given** the match detail page
   **When** I want to connect
   **Then** I see a "Connect" or "Message" button

4. **Given** the match explanation
   **When** it exists
   **Then** it explains why this is a good match

## Tasks / Subtasks

- [ ] Task 1: Create match detail page (AC: #1, #2)
  - [ ] Create `src/app/(protected)/matches/[id]/page.tsx`
  - [ ] Fetch match details from API
  - [ ] Display profile and project info

- [ ] Task 2: Display match analysis (AC: #2, #4)
  - [ ] Show match score prominently
  - [ ] Display explanation text
  - [ ] Highlight skill overlaps

- [ ] Task 3: Add connect action (AC: #3)
  - [ ] Add "Connect" button
  - [ ] Link to messaging (E7)
  - [ ] Show contact info if available
