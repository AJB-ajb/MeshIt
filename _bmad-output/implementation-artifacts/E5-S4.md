# Story 5.4: Build Match Results Page

**Epic:** E5 - Matching Engine  
**Status:** ready-for-dev  
**Priority:** P0  
**Dependencies:** 5.3  
**Blocks:** None

## Story

As a user,
I want to see projects that match my profile,
so that I can find collaboration opportunities.

## Acceptance Criteria

1. **Given** I am logged in
   **When** I navigate to /matches
   **Then** I see projects matching my profile

2. **Given** the match results
   **When** I view each match card
   **Then** I see project title, match score %, and key skills overlap

3. **Given** a match card
   **When** I click on it
   **Then** I am taken to the match detail page

4. **Given** no matches found
   **When** I view the page
   **Then** I see a helpful empty state with suggestions

5. **Given** match results
   **When** I view them
   **Then** they are sorted by score (highest first)

## Tasks / Subtasks

- [ ] Task 1: Create matches page (AC: #1, #5)

- [ ] Task 2: Build match card component (AC: #2)

- [ ] Task 3: Implement navigation (AC: #3)

- [ ] Task 4: Handle empty state (AC: #4)
  - [ ] Design empty state component
