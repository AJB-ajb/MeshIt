# Story 2.3: Build Profile Inference

**Epic:** E2 - GitHub Enrichment  
**Status:** ready-for-dev  
**Priority:** P1  
**Dependencies:** 2.2  
**Blocks:** 2.4

## Story

As a system,
I want to infer skill level and project types from repos,
so that I can enrich the user profile with meaningful data.

## Acceptance Criteria

1. **Given** repo count and star count
   **When** I run inferExperience()
   **Then** I get experience level: beginner (<5 repos), intermediate (5-20), advanced (>20 or >50 stars)

2. **Given** repo descriptions and languages
   **When** I run inferProjectTypes()
   **Then** I get inferred types: Web, Mobile, ML/Data, API, Game, etc.

3. **Given** recent repo activity
   **When** I run inferActivityLevel()
   **Then** I get activity level: low (<3 repos), medium (3-10), high (>10)

## Tasks / Subtasks

- [ ] Task 1: Implement experience inference (AC: #1)
  - [ ] Create `src/lib/github/inference.ts`
  - [ ] Define experience thresholds
  - [ ] Calculate based on repos + stars

- [ ] Task 2: Implement project type inference (AC: #2)
  - [ ] Map languages to project types
  - [ ] Analyze repo descriptions for keywords
  - [ ] Return unique project types

- [ ] Task 3: Implement activity inference (AC: #3)
  - [ ] Count repos updated in last 6 months
  - [ ] Determine activity level

## Dev Notes

- Experience inference is heuristic-based, not perfect
- Project type mapping: JS/TS → Web, Swift/Kotlin → Mobile, Python → ML/Data
- Consider repo descriptions for better inference
