# Story 1.11: Create Basic Layout

**Epic:** E1 - Foundation  
**Status:** ready-for-dev  
**Priority:** P0  
**Dependencies:** 1.1, 1.2  
**Blocks:** All page stories

## Story

As a user,
I want a consistent layout with header and navigation,
so that I can navigate the app easily.

## Acceptance Criteria

1. **Given** I am logged in
   **When** I view any page
   **Then** I see a header with the MeshIt logo
   **And** I see navigation links (Dashboard, Projects, Matches, Messages)

2. **Given** I am logged in
   **When** I look at the header
   **Then** I see my avatar and a dropdown menu
   **And** the dropdown has Profile and Logout options

3. **Given** the layout
   **When** I view on mobile
   **Then** navigation collapses to a hamburger menu
   **And** the menu opens as a drawer/sheet

4. **Given** the layout
   **When** I navigate between pages
   **Then** the layout remains consistent
   **And** active nav item is highlighted
