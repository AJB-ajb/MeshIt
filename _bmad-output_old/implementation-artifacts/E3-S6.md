# Story 3.6: Build Profile Edit Page

**Epic:** E3 - Profile Management  
**Status:** ready-for-dev  
**Priority:** P0  
**Dependencies:** 3.4  
**Blocks:** None

## Story

As a user,
I want to edit my profile after it's created,
so that I can update my skills and availability.

## Acceptance Criteria

1. **Given** I am logged in with a profile
   **When** I navigate to /profile/edit
   **Then** I see my current profile data in editable form

2. **Given** the edit page
   **When** I modify skills, interests, or availability
   **Then** I can save changes
   **And** my profile is updated

3. **Given** I edit my profile
   **When** I save changes
   **Then** my embedding is regenerated
   **And** I see a success message

4. **Given** invalid input
   **When** I try to save
   **Then** I see validation errors
   **And** the save is prevented
