# Story 4.4: Build Project List Page

**Epic:** E4 - Project Management  
**Status:** ready-for-dev  
**Priority:** P0  
**Dependencies:** 4.2  
**Blocks:** None

## Story

As a user,
I want to see all my projects in a list,
so that I can manage them easily.

## Acceptance Criteria

1. **Given** I am logged in
   **When** I navigate to /projects
   **Then** I see a list of my projects

2. **Given** the project list
   **When** I look at each project card
   **Then** I see title, status, expiration date, match count

3. **Given** the project list
   **When** I click on a project
   **Then** I am taken to the project detail page

4. **Given** I have no projects
   **When** I view the list
   **Then** I see an empty state with "Create Project" CTA

5. **Given** a project is expired
   **When** I view the list
   **Then** it shows "Expired" badge with reactivate option
