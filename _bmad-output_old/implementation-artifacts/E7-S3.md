# Story 7.3: Build Conversation List Page

**Epic:** E7 - Messaging  
**Status:** ready-for-dev  
**Priority:** P1  
**Dependencies:** 7.1  
**Blocks:** None

## Story

As a user,
I want to see all my conversations,
so that I can manage my communications.

## Acceptance Criteria

1. **Given** I am logged in
   **When** I navigate to /messages
   **Then** I see a list of my conversations

2. **Given** the conversation list
   **When** I view each item
   **Then** I see the other person's name, avatar, last message preview, and time

3. **Given** a conversation with unread messages
   **When** I view the list
   **Then** it is highlighted with unread indicator

4. **Given** I click on a conversation
   **When** the page loads
   **Then** I am taken to the conversation detail

5. **Given** I have no conversations
   **When** I view the page
   **Then** I see an empty state with helpful message

## Tasks / Subtasks

- [ ] Task 1: Create messages page (AC: #1)
  - [ ] Create `src/app/(protected)/messages/page.tsx`
  - [ ] Fetch conversations from API
  - [ ] Display loading state

- [ ] Task 2: Build conversation list item (AC: #2, #3)
  - [ ] Create `src/components/messages/conversation-item.tsx`
  - [ ] Show avatar, name, preview, time
  - [ ] Highlight unread conversations

- [ ] Task 3: Implement navigation (AC: #4)
  - [ ] Make items clickable
  - [ ] Navigate to /messages/[matchId]

- [ ] Task 4: Handle empty state (AC: #5)
  - [ ] Design empty state
  - [ ] Link to matches page

## Dev Notes

- Conversations are keyed by match_id
- Sort by last_message_at descending
- Consider real-time updates for new messages
