# Story 4.5: Build Project Detail Page

**Epic:** E4 - Project Management  
**Status:** ready-for-dev  
**Priority:** P0  
**Dependencies:** 4.2  
**Blocks:** E5-S4 (View matches)

## Story

As a user,
I want to see full details of a project,
so that I can view matches and manage the project.

## Acceptance Criteria

1. **Given** I navigate to /projects/[id]
   **When** I own the project
   **Then** I see full project details with edit/delete options

2. **Given** the project detail page
   **When** I view it
   **Then** I see title, description, skills needed, commitment, stage, expiration

3. **Given** the project has matches
   **When** I view the page
   **Then** I see a "Matches" section with matched profiles

4. **Given** I click "Edit"
   **When** the edit modal/page opens
   **Then** I can modify project details

5. **Given** I click "Close Project"
   **When** I confirm
   **Then** the project status is set to closed

## Tasks / Subtasks

- [ ] Task 1: Create project detail page (AC: #1, #2)
  - [ ] Create `src/app/(protected)/projects/[id]/page.tsx`
  - [ ] Fetch project by ID
  - [ ] Display all project fields

- [ ] Task 2: Add matches section (AC: #3)
  - [ ] Fetch matches for this project
  - [ ] Display matched profiles as cards
  - [ ] Link to full match view

- [ ] Task 3: Implement edit functionality (AC: #4)
  - [ ] Add Edit button
  - [ ] Create edit modal or navigate to edit page
  - [ ] Save changes via PUT API

- [ ] Task 4: Implement close functionality (AC: #5)
  - [ ] Add "Close Project" button
  - [ ] Show confirmation dialog
  - [ ] Call DELETE API (soft delete)
