# Story 8.7: Animations & Micro-interactions

**Epic:** E8 - UI/UX  
**Status:** ready-for-dev  
**Priority:** P2  
**Dependencies:** 8.1  
**Blocks:** None

## Story

As a user,
I want subtle animations that make the app feel polished,
so that interactions feel responsive and delightful.

## Acceptance Criteria

1. **Given** a page loads
   **When** content appears
   **Then** elements animate in with staggered timing
   **And** the animation is smooth (60fps)

2. **Given** I hover over interactive elements
   **When** I move my cursor
   **Then** there are subtle hover effects
   **And** they feel responsive

3. **Given** I click a button
   **When** the action completes
   **Then** there is visual feedback (scale, color change)

4. **Given** modals or drawers
   **When** they open/close
   **Then** they animate smoothly
   **And** background dims appropriately

## Dev Notes

- Prefer CSS animations for performance
- Use Framer Motion only for complex sequences
- Keep animations subtle (150-300ms duration)
- Respect prefers-reduced-motion

### Animation Utilities

```css
/* tailwind.config.ts */
animation: {
  'fade-in': 'fadeIn 0.3s ease-out',
  'slide-up': 'slideUp 0.3s ease-out',
  'scale-in': 'scaleIn 0.2s ease-out',
}

@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}

@keyframes slideUp {
  from { opacity: 0; transform: translateY(10px); }
  to { opacity: 1; transform: translateY(0); }
}

@keyframes scaleIn {
  from { opacity: 0; transform: scale(0.95); }
  to { opacity: 1; transform: scale(1); }
}
```

### Staggered List Animation

```tsx
// Using animation-delay for stagger
{items.map((item, i) => (
  <div
    key={item.id}
    className="animate-slide-up"
    style={{ animationDelay: `${i * 50}ms` }}
  >
    {item.content}
  </div>
))}
```
