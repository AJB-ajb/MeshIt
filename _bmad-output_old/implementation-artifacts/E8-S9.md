# Story 8.9: Landing Page

**Epic:** E8 - UI/UX  
**Status:** ready-for-dev  
**Priority:** P0  
**Dependencies:** 8.1, 8.2  
**Blocks:** None

## Story

As a visitor,
I want an attractive landing page that explains MeshIt,
so that I understand the value and want to sign up.

## Acceptance Criteria

1. **Given** I visit the homepage
   **When** I am not logged in
   **Then** I see a compelling hero section with value proposition

2. **Given** the landing page
   **When** I scroll
   **Then** I see features, how it works, and social proof

3. **Given** the landing page
   **When** I want to sign up
   **Then** I see clear CTAs (Sign Up with Google/GitHub/etc.)

4. **Given** the landing page
   **When** I view on mobile
   **Then** it looks great and is fully functional

5. **Given** the landing page design
   **When** I view it
   **Then** it feels distinctive and memorable (not generic AI slop)

## Tasks / Subtasks

- [ ] Task 1: Design hero section (AC: #1, #5)
  - [ ] Create compelling headline
  - [ ] Add descriptive subheadline
  - [ ] Add hero image or illustration
  - [ ] Add primary CTA button

- [ ] Task 2: Build features section (AC: #2)
  - [ ] List 3-4 key features with icons
  - [ ] Use grid layout
  - [ ] Add subtle animations

- [ ] Task 3: Build "How it works" section (AC: #2)
  - [ ] Step 1: Create profile
  - [ ] Step 2: AI matches you
  - [ ] Step 3: Connect and collaborate
  - [ ] Use numbered steps or timeline

- [ ] Task 4: Build CTA section (AC: #3)
  - [ ] Add OAuth buttons (Google, GitHub, LinkedIn, Slack, Discord)
  - [ ] Add "Get Started Free" messaging
  - [ ] Make it prominent

- [ ] Task 5: Add footer (AC: #2)
  - [ ] Links: About, Privacy, Terms
  - [ ] Social links if applicable
  - [ ] Copyright

## Dev Notes

- **Avoid generic AI aesthetics**: Be bold with color, typography, layout
- Consider: geometric patterns, gradients, illustrations
- Hero should grab attention in 3 seconds
- Mobile layout is critical - many users will visit on phone

### Design Inspiration

- Bold typography with large headlines
- Geometric background patterns or mesh gradients
- Asymmetric layouts
- Micro-animations on scroll
- Distinctive color palette (from 8.1)
