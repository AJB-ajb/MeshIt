# Story 4.3: Build Project Creation Page

**Epic:** E4 - Project Management  
**Status:** ready-for-dev  
**Priority:** P0  
**Dependencies:** 4.1, 4.2  
**Blocks:** None

## Story

As a user,
I want to create a project by describing it in natural language,
so that I can quickly post what I'm looking for.

## Acceptance Criteria

1. **Given** I am logged in
   **When** I navigate to /projects/new
   **Then** I see a text input for project description

2. **Given** the project creation page
   **When** I enter my description and click "Extract"
   **Then** I see the AI-extracted structured data
   **And** I can review before saving

3. **Given** the extracted project
   **When** I click "Create Project"
   **Then** my project is saved
   **And** I am redirected to the project detail page

4. **Given** extraction errors
   **When** the AI fails to extract
   **Then** I see a helpful error message
   **And** I can try again
